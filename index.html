Luas P. Panjang

Rumus Luas P. Panjang yaitu:
L = P x L

Dimana:
L = Luas
P = panjang
L = Lebar

Hitung Luas P. Panjang

Keliling P. Panjang

Rumus Keliling P. Panjang yaitu:
K = 2 x (P + L)

Dimana:
K = Keliling
P = Panjang
L = Lebar
